lot of improvements including multi monitor support. Look out for the
openSUSE packages on the Build Service X11:XGL repository.

    * There is now 'compiz-decorator' shell script, that tries to

fishes in the cube, 3D plugin now in -extras package.
    * Compiz packages are versioned 0.7 and fusion packages are 0.6.9_git080208
    * This is a development release, it will be followed up by 0.8
stable release
